    GET /v1/health/status¶

    Check API health status

    Description

    Returns a simple message indicating that the API is running and accessible.

    Usage¶

    • Can be used for health checks and monitoring.
    • Requires a valid API key for authentication.

    Response¶

    A JSON object with a welcome message.

    GET /v1/health/info¶

    Get application information

    Description

    Returns detailed information about the application, including name, version, current timestamp, uptime, and environment.

    Usage¶

    • Useful for diagnostics and monitoring.
    • Requires a valid API key for authentication.

    Response¶

    A JSON object with application metadata.

    Input parameters

    Parameter In Type Default Nullable Description
    APIKeyQuery query string N/A No API key
    APIKeyHeader header string N/A No API key
    authorization query string No Chave de autorização, caso não esteja sendo enviada como header.
